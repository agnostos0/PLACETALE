Project Scope & Roles

Scope

A student storytelling portal for sharing place-based tales, exploring stories, and engaging with events and community features.

User Roles

  • Guest: Browse stories/events, view FAQs, read About
  • Student: Register/Login, submit tales, manage profile
  • Admin: Approve/moderate content, manage events/users

Navigation Flow

  1. Guest lands on Home → explores Stories/Events
  2. Guest registers or logs in
  3. Student accesses Dashboard → Submit Tale
  4. Admin reviews submissions → publishes to Stories
  5. Users browse, contact support, read FAQs/Terms
